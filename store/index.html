Hardware


The NXQ® is the next step in desktop PC form factors, it's a small and portable device at 8.7 cm x 8.7 cm x 7.0 cm (WxDxH). It runs on Nitrux® OS and uses KDE 4.13.x. It's built on open-hardware and we've taken the extra step to ensure a great build quality for it despite its affordable price-tag. It runs on our preferred Linux environment, the KDE Software Compilation. KDE is one of the most popular desktop environments in Linux, it's incredibly packed with features and performance. With the NXQ® you can carry your own desktop with you, meetings, presentations all are simplified by just hooking up the device to an HDMI capable screen, plug-in your wireless accessories and you'll be able to access your files just like you would at home.


Specifications

  • Samsung Exynos 4412 (4-core, 2.0 GHz, 2 GB RAM, Mali 400 MP4)
  • Ethernet, 3x USB 2.0, micro-USB, micro-HDMI and a 3.5 mm audio jack
  • Nitrux® OS w/ KDE 4.13.x

The Nitrux KDE Suite is single package that contains both a Plasma and a QtCurve theme for KDE that enchances the look of your KDE desktop and stand out from the rest!. Licensed under the Creative Commons Attribution- NonCommercial-NoDerivatives International 4.0 License.


First step: Purchase the theme suite

  • Click the arrow to show the menu

      • Package includes the following:

      • Nitrux® Plasma themes
      • Nitrux® QtCurve theme
      • Nitrux® Color scheme
      • Nitrux® Exclusive wallpapers

Second step: Select both themes in KDE

  • Click the arrow to show the instructions

    First you've got to install QtCurve from your distribution repositories. The steps below are specifically for Ubuntu and its derivatives. Regardless of them you can find QtCurve in most distributions, the package is usually called the same.


    Install QtCurve from the Terminal (Ubuntu and derivatives).

    sudo apt-get install qtcurve #Install QtCurve, this is very important

    After installing QtCurve go to the KDE System Settings>Application Appearance then go to Style click on the Applications tab and on the widget style dropdown menu select QtCurve. And then Click on configure.


    When the Configure QtCurve shows up click on import. Then simply select the Nitrux.qtcurve file located in the directory where you downloaded the suite. Then select it from Presets the dropdown menu in the same window.


    To use the Nitrux color scheme go back to Application Appearance and go to Colors, then click on import and choose the Nitrux.colors file and click apply to observe the changes.


    To use the Plasma theme you will have to move the Nitrux Plasma folder to this path: /home/$USER/.kde/share/apps/desktoptheme/. Then in the KDE System Settings go to Workspace Appearance and then to Desktop Theme there you can select the Nitrux Plasma theme.
